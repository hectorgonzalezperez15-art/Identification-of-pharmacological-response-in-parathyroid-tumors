# Identificaci-n-de-respuesta-farmacol-gica-en-tumores-paratiroideos
Calcium metabolism is regulated by parathyroid hormone (PTH). Parathyroid tumors overproduce PTH, causing hypercalcemia. This study uses RNA-seq on 24 ERα− primary cultures treated with DPN and OHT to identify transcriptomic changes and estrogen-regulated pathways involved in proliferation, apoptosis, and hormone secretion.
